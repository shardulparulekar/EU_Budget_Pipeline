# Data Files

Place your Excel files here:

- `eu_budget_spending_and_revenue_2000-2023.xlsx` (required)
- `eu_budget_spending_and_revenue_2000-2022.xlsx` (required)

## Where to Get Data

Download from: [European Commission Budget Database](https://ec.europa.eu/info/strategy/eu-budget_en)

OR contact repository owner for sample data

OR these are already available in the same folder of this file

## File Requirements

- **Format:** Excel (.xlsx)
- **Size:** ~1-2 MB
- **Years:** 2000-2022 and 2000-2024 (25 sheets, one per year)
- **Columns:** Country codes, budget headings, amounts
